What It Does

Project Overview

Kolam is a traditional South Indian art form — intricate geometric patterns drawn at dawn with rice flour at home thresholds. This project digitizes, preserves, and extends that heritage using modern machine learning, computer vision, and speech technology.

Built as a multi-mode Streamlit web app, it lets you load, draw, upload, or conversationally generate Kolam patterns — and analyze the deep mathematics hiding within each one.

Creation Modes

Application Modes

I

Predefined Kolam

Load one of three bundled patterns (Simple, Lotus, Star). Binarized via Otsu's thresholding, then skeletonized to extract a point-cloud for transformation and analysis.

II

Upload Image

Upload any Kolam photograph. The CV pipeline decodes, grayscales, thresholds, and skeletonizes it into an editable point cloud.

III

Draw Freehand

Draw directly on an interactive canvas. Optionally overlay a reference image for tracing. Apply the neon glow effect for a stunning visual finish.

IV

Kaleidoscope Draw

Draw with 8-fold rotational symmetry in a built-in HTML5 canvas. Choose from 9 solid colors or rainbow mode. Touch-enabled for mobile and tablet.

V

Conversational Drawing Accessibility

Tell Chitra what to draw using natural speech. Designed for visually impaired users — pure voice, zero pointing device required.

VI

Generative Design AI-Powered

Tweak symmetry, complexity, and style sliders, then click "Generate". The algorithm creates entirely new Kolam designs using B-spline interpolation and rotational math.

Voice Assistant

Chitra — Your AI Companion

Chitra is the soul of this app. A personal voice assistant with an Indian English female voice (via Google TTS tld='co.in'), she processes speech with Google Speech Recognition and a Sphinx offline fallback — meaning she works without internet when needed.

Every action in the app is accessible via voice command, and every Chitra response is simultaneously displayed as text and spoken aloud — audio plays in a background thread so the UI never freezes.

Get Started

Installation & Setup

Clone the repository

git clone https://github.com/your-username/Kolam-and-Kaleidoscope-Designer-with-Chitra.git
cd Kolam-and-Kaleidoscope-Designer-with-Chitra

Install dependencies

pip install streamlit opencv-python scikit-image scipy networkx \
            streamlit-mic-recorder SpeechRecognition gtts playsound==1.2.2 \
            svgwrite matplotlib streamlit-drawable-canvas Pillow

Update image paths in kolam.py

# In kolam.py, ensure the paths point to the same folder:
PREMADE_PATTERNS = {
    "Simple Kolam": "simple kolam.jpg",
    "Lotus Kolam":  "lotus kolam.jpg",
    "Star Kolam":   "star kolam.jpg"
}

Run the app

streamlit run kolam.py
# Open http://localhost:8501 in your browser

Cultural Context

About Kolam

Kolam is a traditional South Indian art form practiced primarily in Tamil Nadu — intricate geometric patterns drawn at dawn with rice flour at the threshold of homes. Far beyond decoration, each Kolam embodies deep mathematical principles: Topology, Graph Theory, Symmetry Groups, and Fractal Geometry. This application helps preserve and study this beautiful mathematical art while making it accessible to everyone — through natural conversation, voice, and the quiet magic of an algorithm drawing what hands once drew in flour on stone.
